Exam Instructions & Rules

General Guidelines:

1. The examination contains exactly 200 multiple-choice questions. Total marks is 100.

2. Each correct answer carries 0.5 marks. There is NO negative marking for incorrect responses.

3. Total duration of the examination is 3 Hours (180 Minutes). The timer is displayed in the top bar.

4. The test environment automatically switches the browser to Fullscreen Mode. Minimize window toggles or exiting fullscreen may result in a system flag.

5. Do not close or refresh the tab during the test. If done accidentally, the local storage persistence will allow resuming, but candidate must proceed quickly.
