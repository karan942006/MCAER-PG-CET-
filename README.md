## ✨ Features

<div align="center">

| 🎯 Feature | 📝 Description |
|:---:|:---|
| 📋 **295+ Questions** | Curated from MCAER PG CET syllabus 2012–2025 |
| 🏫 **6 Subjects** | Agri Engineering, Horticulture, Genetics, Pathology, Soil Science, Economics |
| 🎲 **Smart Shuffling** | Questions and options randomly shuffled every test |
| ⏱️ **3-Hour Timer** | Real countdown timer matching actual CET duration |
| 🎨 **Question Palette** | CET/JEE style — Grey / Red / Green / Purple states |
| 📊 **Analytics** | Speedometer gauge + subject-wise performance bars |
| 💾 **Auto-Save** | Resume interrupted tests from browser localStorage |
| 🖨️ **Print Result** | Export scorecard as PDF directly from browser |
| 🔐 **Admin Portal** | Password-protected admin console with JSON import |
| 📱 **Responsive** | Works on Desktop, Tablet & Mobile |

- ✅ **200 questions** per mock test (randomly selected from 295+ database)
- ✅ **0.5 marks** per correct answer → Total **100 marks**
- ✅ **No negative marking**
- ✅ **3 hours** duration
- ✅ Fullscreen mode enforced during exam
